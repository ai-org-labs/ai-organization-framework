# AOF v7.6.0 Release Notes

`v7.6.0` is the Provider-Neutral Session Export release.

It follows `v7.5.0` Requirement Coverage. v7.5 made progress and forecast claims auditable. v7.6 adds portability: AI session evidence can be packaged without depending on one provider stream format as the source of truth.

## What Changed

- Added `session-export-record`.
- Added `session-export-audit`.
- Added schemas for provider-neutral Session Export and Session Export audit results.
- Integrated `session-export-audit` into `release-state-audit` for `7.6.0` and later releases.
- Added focused runtime tests proving missing exports fail and complete exports pass.
- Added TASK-095 release evidence for readiness, context integrity, session evidence, actor roster, actor handoffs, Council review, Work Execution Packet, Multi-Actor Pilot, Parallel Lane Pilot, Requirement Coverage, Archmap impact, and Session Export audit.

## Boundary

`v7.6.0` proves structural/runtime export governance evidence.

It does not prove:

- semantic correctness
- universal privacy safety
- provider API integration
- real-time stream ingestion
- external adoption

## Post-v7.6 Direction

Recommended sequence:

1. `v7.7`: Adoption-Grade v7 Runtime
2. `v7.8`: Mission Control Coverage Forecast Projection
3. `v8.0`: Externalized Organization Runtime
