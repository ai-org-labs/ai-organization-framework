## Release Decision Boundary

Release only if v9.5 makes the provider track safer to understand:

- a controlled execution candidate is not enough
- a provider failure path must be reconstructable
- recovery evidence must include detection, containment, rollback, notification, learning, and resume/stop governance
- production execution remains unauthorized
- audit pass does not claim live production recovery performance
