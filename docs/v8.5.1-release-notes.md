# AOF v8.5.1 Release Notes

`v8.5.1` is a release-clarity patch for `v8.5.0`.

It does not expand external execution authority. It fixes entrypoint and
operator-facing wording so the public release surface matches the actual
runtime boundary.

## Fixes

- Updated README stable release and current release links from `v8.2.0` to
  `v8.5.0`.
- Updated Quickstart install commands from `v7.0.0` to `v8.5.0`.
- Changed Mission Control external runtime safety projection wording from a
  broad `safe` status to `governance_boundary_ready`.
- Added explicit `production_execution_safety_status: not_proven` to the
  Mission Control projection.

## Boundary

`v8.5.1` proves release-surface consistency and clearer operator wording only.
It does not prove production provider execution safety, credential safety,
billing safety, provider correctness, market value, or third-party adoption.
