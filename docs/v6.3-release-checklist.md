## Sign-Off Rule

Do not sign off because AOF has more commands or more documentation.

Sign off only when:

- command help is registry-backed
- AI can get compact command routing help without loading the full CLI reference
- command help states failure meaning and QIF boundary
- QIF v0.3.0 is recorded as an external provider profile
- QIF v0.3.0 Discovery Layer is not overclaimed as executable verifier support
- semantic uncertainty remains visible
