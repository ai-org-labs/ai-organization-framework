## Release Decision Boundary

Release only if v9.6 makes provider execution spend safer to understand:

- a controlled execution candidate is not enough
- an incident recovery drill is not enough
- provider execution must declare budget owner, cost ceiling, token/call/retry caps, quota, rate limit, billing, overage policy, and stop conditions
- production execution remains unauthorized
- audit pass does not claim real provider billing correctness or actual quota enforcement
