# AOF v8.9.0 Release Notes

`v8.9.0` ships External Runtime Outcome Evidence and Learning Loop.

## What Changed Since v8.8

- Added provider outcome evidence records and audit.
- Added provider learning-loop records and audit.
- Added v8.9 release-state gates for outcome and learning evidence.
- Added Mission Control projections for outcome and learning-loop status.
- Added self-hosting TASK-109 evidence that links the approved/reproduced/rollback-ready TASK-107 provider preflight path to bounded outcome evidence and a learning-loop update.

## Operator Meaning

AOF no longer stops at "this external runtime action was approved, reconstructable, and rollback-ready." It now requires a separate record of what was observed and what the organization learned from that observation.

## Boundary

This release does not perform production external writes.

It proves bounded outcome traceability and learning-loop update evidence from canonical artifacts only.
