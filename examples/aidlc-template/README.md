# AIDLC Example Template

This template demonstrates an AOF project for software-style AIDLC work.

Seeded:

- actors: Visionary, Builder, Guardian
- workflow: `.aof/workflows/aidlc.yaml`
- governance and policy files
- decision record template and schema
- sample sessions
- sample signal

Generated during use:

- new sessions
- decision records
- council execution packets
- outcome reports
- verification bundles

Start here:

```bash
node ../../src/cli.js run "アプリAをどう収益化するべきか" --project .
```

Then follow `../../docs/quickstart.md`.
